# Taiex_analysis
Taiex_analysis

![](https://i.imgur.com/FRYKJQk.png)

以Python編寫台股個股分析程式，可抓取個股過去的資料，用於程式化交易策略回測。

## 主要功能：
1) 爬取指定個股歷史交易資料
2) 圖像化個股歷史交易資料
3) 設定程式化交易策略套用至指定個股 (編寫中)
4) 程式化交易績效分析 (編寫中)
